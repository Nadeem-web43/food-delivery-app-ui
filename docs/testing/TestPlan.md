# Test Plan

## Project

Food Delivery App UI/UX Design

## Objective

The purpose of testing is to evaluate the usability, functionality, and user experience of the Food Delivery App.

## Testing Type

- Usability Testing
- UI Consistency Testing
- Navigation Testing
- Accessibility Testing

## Test Devices

- Android Smartphone
- iPhone
- Desktop (Figma Preview)

## Test Areas

- Login
- Navigation
- Search
- Cart
- Checkout
- Payment
- Order Tracking
- Profile

## Expected Result

Users should be able to complete food ordering quickly and easily without confusions.
