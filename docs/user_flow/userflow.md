# User Flow

## Project

Food Delivery App UI/UX Design

## Objective

The user flow illustrates the journey a user follows while using the Food Delivery App.

## Main User Journey

1. Open Application
2. Splash Screen
3. Login or Sign Up
4. Home Screen
5. Search Food or Restaurant
6. View Restaurant Details
7. Select Food Item
8. Add Food to Cart
9. View Cart
10. Checkout
11. Select Payment Method
12. Confirm Order
13. Track Order
14. View Order History
15. Logout
