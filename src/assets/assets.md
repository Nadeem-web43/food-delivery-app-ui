# Assets

## Overview

This folder contains all visual assets used in the Food Delivery App UI/UX project.

These assets are used while designing the application screens in Figma.

---

## Asset Categories

### Logo

Application logo and app icon.

### Food Images

Images of food items such as:

- Pizza
- Burger
- Biryani
- Pasta
- Sandwich
- Noodles
- Fried Rice
- Desserts
- Beverages

### Restaurant Images

Images and logos of restaurants.

Examples:

- Pizza Hut
- Domino's
- KFC
- McDonald's
- Burger King
- Local Restaurants

### Banners

Promotional banners for:

- Discounts
- Offers
- Free Delivery
- Combo Deals

### Illustrations

Illustrations used in:

- Splash Screen
- Empty Cart
- Order Success
- No Internet
- No Search Results

### Profile Images

Default profile pictures used for demonstration.

### Backgrounds

Background images used throughout the application.

---

PNG

JPG

SVG

## Purpose

These assets improve the visual appearance of the application and create a modern user experience of UI/UX
