# Components

## Overview

This folder contains all reusable UI components used in the Food Delivery App UI/UX project.

Using reusable components helps maintain consistency, improves design efficiency, and makes updates easier.

---

## Components Included

### Buttons
Primary, Secondary, Outlined and Disabled buttons.

### Input Fields
Email, Password, Search and Address input fields.

### Search Bar
Search component used on the Home and Search screens.

### Navigation
Top App Bar and Bottom Navigation Bar.

### Food Cards
Cards displaying food image, name, price, rating and Add button.

### Restaurant Cards
Cards displaying restaurant image, name, rating, delivery time and distance.

### Category Chips
Food category selection chips.

### Rating Badge
Displays restaurant and food ratings.

### Cart Item
Displays selected food items in the cart.

### Payment Card
Displays payment methods.

### Profile Card
Displays user profile information.

### Order Card
Displays previous orders.

### Address Card
Displays saved delivery addresses.

### Coupon Card
Displays discount coupons.

### Offer Banner
Displays promotional offers.

### Dialogs
Confirmation and success pop-up dialogs.

---

## Purpose

These components are reused across multiple screens to provide a consistent user experience.
