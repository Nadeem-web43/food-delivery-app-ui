The **Food Delivery App UI/UX Design** project was developed as part of the **CodeZoner UI/UX Internship**.

The objective of this project is to design a modern, user-friendly, and visually appealing mobile application that enables users to browse restaurants, order food, make secure payments, and track their orders in real time.

The project follows the complete **UI/UX Design Process**, including research, user flow, wireframing, design system creation, high-fidelity UI design, prototyping, usability testing, and final documentation.
